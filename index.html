🚌 1-hour commute study plan

0–10 min: House White + House Red: names, countries, prices.

10–25 min: Aroma Quiz — repeat descriptions until they feel easy.

25–40 min: Fresh/Crisp Whites + Round/Fruity Reds.

40–50 min: Price Drill — learn glass/bottle prices.

50–60 min: Speed Round + quick review of videos/notes.

Mini memory rule

Name → country → grape → 3 taste words → price. Example: Villa Del Lago → Italy → Pinot Grigio → dry/apple/balanced → €8.5/€31.5.
